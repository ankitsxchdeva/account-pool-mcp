account-pool-mcp

Exclusive, crash-safe leases on a pool of test accounts — so parallel agent QA sessions never grab the same one.
› SQLite-backed atomic allocation
› per-lease TTL auto-reclaim on crash
› no shared parent process
$ npx -y account-pool-mcp
account-pool
$ account-pool lease realtor --holder A
realtor_01  leased ✓
$ account-pool lease realtor --holder B
realtor_02  leased ✓  different account
$ account-pool lease realtor --holder C
pool exhausted — refused  no double-booking
